# Baby Lion Productions & Device Limited

Landing page for Baby Lion Productions - a full-service video production company based in Nigeria.

## Tech Stack

- React + Vite
- HMR with [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react)

## Development

```bash
npm install
npm run dev
```

## Buil

```bash
npm run build
```
